Описание продукта
Образовательный контент представляет собой банк интерактивных заданий с автоматической проверкой и генерацией условий по разделу «Элементы векторной алгебры» для суворовцев 9-11 класса. 
Интерактивные задания позволяют значительно повысить мотивацию и вовлеченность учащихся, обеспечивают объективную оценку результатов и способствуют эффективному освоению учебного материала. Данный проект решает проблему недостаточной практики учащихся, способствует развитию самостоятельности и критического мышления.
Целями разработанного банка заданий является формирование глубоких и прочных знаний по геометрии; повышение эффективности процесса обучения путем внедрения современных образовательных технологий.
Анализ задачного материала способствовал разработке авторских типов заданий и их педагогического дизайна. При создании учитывались педагогические цели и методики обучения, типовые задания, а также возможности технической реализации. Формирование банка заданий осуществлялось с использованием следующих языков программирования: HTML, CSS, JavaScript.  При разработке некоторых заданий за основу были взяты простейшие игровые механики: раскраска, найди пару (тройку), рисование фигур, графический диктант.
